## Задача 1

На складе лежат разные фрукты в разном количестве.
Нужно написать функцию, которая на вход принимает любое количество названий фруктов и их количество, а возвращает общее количество фруктов на складе


```python
def calculate_total_fruits(**fruits):
    total = 0
    for fruit, quantity in fruits.items():
        total += quantity
    return total
```


```python
total_fruits = calculate_total_fruits(apples=10, oranges=15, bananas=20)
print(total_fruits)
```

    45


## Задача 2

Дан список с затратами на рекламу. Но в данных есть ошибки, некоторые затраты имеют отрицательную величину. Удалите такие значения из списка и посчитайте суммарные затраты
[100, 125, -90, 345, 655, -1, 0, 200]
Используйте list comprehensions


```python
expenses = [100, 125, -90, 345, 655, -1, 0, 200]

positive_expenses = [expense for expense in expenses if expense >= 0]
total_expenses = sum(positive_expenses)

print(total_expenses)
```

### Подзадача 3.2

Найдите выручку компании в зависимости от месяца.
Для этого напишите функцию, которая на вход принимает список с датами и список с выручкой, а на выходе словарь, где ключи - это месяцы, а значения - это выручка.
Используйте аннотирование типов.


```python
from typing import List, Dict

def calculate_revenue_by_month(dates: List[str], amounts: List[int]) -> Dict[str, int]:
    revenue_by_month = {}  # Создаем пустой словарь для хранения выручки по месяцам
    for date, amount in zip(dates, amounts):
        month = date[5:7]  # Извлекаем месяц из даты
        if month in revenue_by_month:
            revenue_by_month[month] += amount
        else:
            revenue_by_month[month] = amount
    return revenue_by_month

dates = ['2021-09-14', '2021-12-15', '2021-09-08', '2021-12-05', '2021-10-09', '2021-09-30', '2021-12-22', '2021-11-29', '2021-12-24', '2021-11-26', '2021-10-27', '2021-12-18', '2021-11-09', '2021-11-23', '2021-09-27', '2021-10-02', '2021-12-27', '2021-09-20', '2021-12-13', '2021-11-01', '2021-11-09', '2021-12-06', '2021-12-08', '2021-10-09', '2021-10-31', '2021-09-30', '2021-11-09', '2021-12-13', '2021-10-26', '2021-12-09']
```
